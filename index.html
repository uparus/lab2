Команды Git
Команда Описание
git init Инициализирует новый локальный репозиторий Git
git clone url Копирует существующий репозиторий на локальный компьютер.
git add file Добавляет содержимое указанного файла
git add . Добавляет изменения из рабочей директории в индекс
git commit -m "comment" Создаёт коммит (снимки изменений) в локальном репозитории проекта с указанным комментарием.
git commit -a -m "comment" Создаёт коммит всех отслеживаемых изменений и добавляет к нему указанный комментарий.
git push url Синхронизирует локальные коммиты с удалённым репозиторием
git status Отображает состояние рабочего каталога и области подготовленных файлов
git log Позволяет просматривать историю коммитов в репозитории
git branch branch_name Создаёт новую ветку.
git checkout branch_name Позволяет переключиться на указанную ветку
git checkout -b branch_name Создаёт новую ветку с указанным именем и автоматически переключается на неё
git remote Позволяет управлять связями между локальным репозиторием и другими версиями кода
